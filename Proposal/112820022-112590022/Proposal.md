# Game Introduction
Untitled Action是一個2D橫向冒險類遊戲，從造物主是外星人，幾億年前留下DNA在地球起，孕育出人類，而外星人"元素人類"殖民了許多星球，隨著牠們人口漸增，現有的星球都不夠用了，於是打算奪取地球以及DNA留在地球的實驗結果，因此在兩方戰爭下，人類因戰火大量減少。人類們私下執行了多項計畫，其中一項計畫正是創造出我們主角與其他特殊戰鬥體突破前線直入敵營企圖瓦解對方。

主角Marcelina由半個人工智慧的Marce（玩家）與半個人類本體的Celina（主角），為了防止記憶動搖16歲Celina的判斷設計上特意將過去的記憶上鎖，兩者可以在心裡互相溝通，動作上是由Marce負責，話語權是在Celina，玩家可以透過Marce與Celina的對話潛在影響Celina對於之後劇情的發展走向。

## 基本
* ### 主角
    + 血量：歸零則死亡，死亡回到上一個存檔點，能量都歸零
    + 能量：能量衝能滿後可以施放大招
    + 攜帶：３格欄位，裝備後可獲得該攜帶物的屬性加成與特殊效果
    + 武器：劍類（近程）、槍類（遠程）、特殊（勾爪），可隨著劇情發展或地圖探索獲得並更換選擇
    + 大招武器：可隨著劇情發展或地圖探索獲得並更換選擇
    + 閃避：無敵幀（幀數待訂）並衝刺一段
    + 攻擊：拿手上武器進行段數攻擊
    + 技能：施展手上武器的技能攻擊
    + 大招：施展裝備上的大招武器，能量歸零（可隨著劇情發展或地圖探索獲得並更換選擇）
* ### NPC
    + 互動：基礎對話／觸發劇情／劇情動畫／特殊系統（像是交易系統）
* ### 小怪
    + 血量：歸零則死亡
    + 攻擊：角色一般攻擊
    + 硬值：受到打擊則減少，降至零時則昏厥一段時間，昏厥結束後會補滿
* ### 菁英怪
    + 血量：歸零則死亡
    + 攻擊：角色一般段數攻擊
    + 技能：角色技能攻擊（數量待訂）
    + 硬值：受到打擊則減少，降至零時則昏厥一段時間，昏厥結束後會補滿
* ### Boss
    + 血量：歸零則死亡
    + 能量：能量衝能滿後可施放大招
    + 攻擊：角色一般段數攻擊（數量待訂）
    + 技能：角色技能攻擊（數量待訂）
    + 大招：施放角色大招（依特殊判斷情況）
    + 硬值：受到打擊則減少，降至零時則昏厥一段時間，昏厥結束後會補滿
* ### 協同（限於特殊場景或劇情）
    + 血量：歸零則死亡
    + 能量：能量衝能滿後可以施放大招
    + 攻擊：角色一般攻擊
    + 技能：角色攻擊技能
    + 大招：角色大招
## 系統
* ### 戰鬥系統
    打擊掉幀（幀數待訂）、抗性攻擊可以有效減少硬值
* ### 情感系統
    透過選擇與Celina的對話增減不同陣營的數值影響往後劇情走向
* ### 快捷選單武器系統
    按下tab有以主角為圓心的選單，並減緩畫面呈現時間直到玩家關閉（再一次tab）或選擇
* ### 交易系統
    販賣屬性靈魂獲得報酬
* ### 背包系統
    武器、攜帶物、屬性靈魂等
* ### 存檔與轉移系統
    玩家到休息點可以儲存紀錄或是傳送至過去開闢過的休息點
* ### 選單系統+UI
    設定頁面（Esc）、背包（B連結背包系統）、地圖（M連結地圖系統）
* ### 地圖系統
    開闢過的地圖區塊顯示，未去過的門以特殊記號顯示
* ### 作弊系統
    我是外掛：P

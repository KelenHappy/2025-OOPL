# Game Introduction
像素風魂系遊戲，玩家及boss都只有1條血，玩家需一邊閃避攻擊一邊找出boss的弱點
攻擊方式可以靠射出箭矢及回收箭矢兩種。
我們預計從原遊戲中挑選6隻我們覺得特性有趣的boss來實作，包含boss技能和場地特性。

影片連結: https://www.youtube.com/watch?v=rtjXDVIKrNw&t=97s

# Game Introduction

我們的遊戲名為「超級瑪莉歐兄弟」，目的是希望能夠複刻這款經典的遊戲，內容的所有物件都是從網路上下載下來。遊戲中，我們會設置各種不同的場景關卡，過程如果碰到怪物、有害障礙物或跳入虛空、岩漿等，便會失去一條命，每一關總共有三條命。

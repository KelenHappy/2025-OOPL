# Game Introduction
    困難版馬力歐，每個關卡有不同的地形需要靠玩家操作來通過，並且會包含一些需要解謎互動的部分，並且每關最後會包含一個Boss關卡
# 製作順序
    製作地圖->角色能在地圖移動->製作Boss->按照第一關的方式製作後面的關卡
